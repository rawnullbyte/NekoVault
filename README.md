# NekoVault
## Install requirements for server
```bash
cd core
pip install -r requirements.txt
```

## Install requirements for CLI client
```bash
cd cli
pip install -r requirements.txt
```

## Running server and CLI client
### Terminal 1
```bash
cd core
python3 main.py
```
### Terminal 2
```bash
cd cli
python3 main.py
```
